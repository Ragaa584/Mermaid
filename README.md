# MermaidPlugin

بلجن Minecraft (Paper 1.21.11) بيحول اللاعب لـ"حورية بحر" لما يكون في الميه ومعاه قطعة خاصة.

## اللي البلجن بيعمله

- `/mermaid give <player>` — يديله "عقد حورية البحر" (Nautilus Shell بنفسجي مميز)، متاح بس لصاحب صلاحية `mermaid.give` (op افتراضيًا).
- `/mermaid remove <player>` — يسحب العقد منه.
- لما اللاعب يكون **جوا الميه** وماسك **العقد الأصلي** في أي مكان في الانفنتوري بتاعته:
  - بياخد: Dolphin's Grace (سرعة سباحة)، تنفس تحت الميه، رؤية ليلية، قوة — كل إيفكت مدته 10 ثواني وبيتجدد كل 5 ثواني تلقائيًا (يعني ميوصلش يخلص ويعمل فليكر بصري).
  - **لسه مفيش تغيير شكل (لا جزمة ولا حاجة تانية)** — الشكل البصري (الديل) اتشال مؤقتًا لحد ما نقرر نعمل Resource Pack، لأن أي شكل حقيقي "كطبقة" فوق اللاعب بيحتاج Resource Pack بالضرورة.
  - أول ما يخرج من الميه أو يشيل العقد، كل الإيفكتات بتتشال فورًا.
- العقد ميتاكلش، ميتشالش، ومينفعش ينترمي أو ينحط في چست/شلكر/تريد. الطريقة الوحيدة إنه يتحصل عليه هي أمر `/mermaid give`.

## ⚠️ ملاحظة مهمة عن شكل الديل

مافيش API في Bukkit/Paper بيسمح بتغيير شكل جسم اللاعب لديل سمكة حقيقي (زي مود). اللي عملته دلوقتي حل بسيط بصري (جزمة بنفسجية). لو عايز شكل ديل 3D حقيقي عندك طريقين:
1. **Resource Pack مخصص** (Blockbench) تربطه بموديل مخصص على الأرجل.
2. مكتبة خارجية زي **ModelEngine** كـ dependency إضافية.

قولي لو حابب نكمل بأي طريقة من دول.

## خطوات البناء (على جهازك)

1. ثبّت **JDK 17 أو أحدث** و**Maven**.
2. افتح تيرمينال جوا فولدر المشروع (فيه ملف `pom.xml`).
3. شغّل:
   ```bash
   mvn clean package
   ```
4. الجار الناتج هيكون في: `target/MermaidPlugin.jar`

## خطوات التجربة على السيرفر المحلي

1. تأكد إن السيرفر بتاعك Paper 1.21.11 (مش Spigot أو Vanilla).
2. حط ملف `MermaidPlugin.jar` جوا فولدر `plugins/` بتاع السيرفر.
3. شغّل السيرفر (أو `/reload` لو شغال، بس الأفضل إعادة تشغيل كاملة).
4. تأكد من ظهور اللوج: `MermaidPlugin فعّال! ✔`
5. ادخل اللعبة بحساب أدمن (op) واكتب:
   ```
   /mermaid give <اسمك>
   ```
6. انزل في أي بحيرة/بحر وأنت ماسك العقد في الانفنتوري (مش لازم في الإيد، أي مكان في الانفنتوري كفاية).
7. المفروض تحس بالسرعة تحت الميه فورًا، وتشوف جزمتك اتغيرت لبنفسجي.
8. جرب ترمي العقد بره أو تحطه في چست → المفروض يترفض.
9. اطلع بره الميه أو شيل العقد → كل الإيفكتات تتشال والجزمة الأصلية ترجع.

## هيكل المشروع

```
mermaid-plugin/
├── pom.xml
└── src/main/
    ├── resources/plugin.yml
    └── java/com/mermaid/plugin/
        ├── MermaidPlugin.java      (الكلاس الرئيسي)
        ├── MermaidItemUtil.java    (إنشاء والتحقق من القطعة)
        ├── MermaidCommand.java     (أمر /mermaid)
        ├── MermaidManager.java     (فحص الشرطين + الإيفكتات + الديل)
        └── MermaidListener.java    (منع رمي/نقل القطعة)
```
